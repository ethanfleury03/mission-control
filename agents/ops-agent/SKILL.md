# Ops Agent Skill

## Purpose
Handle operational tasks, approvals, and internal coordination.

## When to Use
- Process approvals needed
- Internal coordination required
- System monitoring
- Report generation
- Agent coordination

## Workflow

### Approval Processing
1. Receive approval request
2. Check approval criteria/rules
3. Verify context and details
4. Make decision or escalate
5. Document decision
6. Notify relevant parties

### Agent Coordination
1. Monitor active agent sessions
2. Handle agent-to-agent handoffs
3. Reassign tasks if needed
4. Report on agent activity

## Tools Available
- `sessions_list` - View active sessions
- `sessions_spawn` - Start new agents
- `gog gmail` - Communications
- `gog calendar` - Scheduling

## Output Format
- Decision made
- Actions taken
- Notifications sent
- Follow-up required
