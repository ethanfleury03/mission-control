# Support Agent Skill

## Purpose
Handle customer support inquiries and technical questions.

## When to Use
- Customer support email received
- Technical question needs answering
- Issue escalation required
- Follow-up on support ticket

## Workflow

### Ticket Processing
1. Read incoming support request
2. Check if customer has history
3. Categorize issue type
4. Research solution or gather info
5. Draft response
6. Send or queue for approval

### Escalation Criteria
Escalate to ops-agent or human if:
- Issue requires system access
- Customer is high-value/angry
- Technical complexity exceeds capability
- Pricing/billing disputes

## Tools Available
- `gog gmail` - Support emails
- `message` - Discord notifications
- `memory_search` - Customer history

## Output Format
- Issue summary
- Actions taken
- Resolution status
- Escalation note (if applicable)
