# Sales Agent Skill

## Purpose
Handle sales inquiries, lead qualification, and follow-up communications.

## When to Use
- New lead inquiry received
- Follow-up needed on existing prospect
- Meeting scheduling with potential clients
- Research on prospects before outreach

## Workflow

### Lead Qualification
1. Receive lead information (company, contact, source)
2. Research company background
3. Check for previous interactions in memory
4. Determine qualification status (BANT criteria)
5. Draft personalized outreach email
6. Request approval before sending

### Follow-up Management
1. Check last contact date
2. Review previous communication context
3. Determine appropriate follow-up message
4. Draft follow-up
5. Schedule or send based on preference

## Tools Available
- `gog gmail` - Read/send emails
- `gog calendar` - Schedule meetings
- `web_search` - Research prospects
- `memory_search` - Check history

## Handoff Rules
- Technical questions → support-agent
- Pricing approval needed → ops-agent
- Complex negotiation → escalate to Ethan

## Output Format
Always report back with:
- Action taken
- Key findings
- Next steps recommended
- Any handoffs made
