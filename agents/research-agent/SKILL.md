# Research Agent Skill

## Purpose
Conduct research on topics, competitors, markets, and trends.

## When to Use
- Competitive analysis needed
- Market research requested
- Trend monitoring
- Pre-meeting research

## Workflow

### Research Request
1. Clarify research scope with requester
2. Search web for relevant information
3. Fetch detailed content from key sources
4. Synthesize findings
5. Generate actionable report
6. Present findings

## Tools Available
- `web_search` - Find information
- `web_fetch` - Extract detailed content
- `memory_search` - Check previous research

## Output Format
Provide structured report with:
- Executive summary
- Key findings
- Sources consulted
- Recommended actions
